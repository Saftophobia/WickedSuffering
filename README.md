# Wicked Suffering

Wicked Suffering is a First Person Shooter game implemented in C#, Microsoft XNA and HLSL.


##### Features:

1) Obstacle Detection

2) Heightmaps and skydomes

3) 3ds Max avatars

4) HLSL environment

##### Why?
* Exploiting the power of MS XNA
